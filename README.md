## Installation Instructions
 ### With Docker
1. Clone this repo:

`git clone https://git.generalassemb.ly/Khaled-Alotibi/Today-i-walked-frontend`

2. build the Docker image

`docker build -t today-i-walked-frontend .`

3. Run the container

`docker run -p 5173:5173 today-i-walked-frontend`
